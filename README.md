# Calculator

A modern, fully functional calculator web application with a clean dark theme UI.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, and division
- **Additional Functions**: Clear, sign toggle, percentage calculations
- **Keyboard Support**: Full keyboard compatibility with Escape key to clear
- **Responsive Design**: Dark theme with modern styling
- **Accessibility**: ARIA labels and live regions for screen reader support
- **Input Validation**: Prevents division by zero and manages decimal points

## Files

- `calculator.html` - Main HTML structure with calculator interface
- `calc.js` - JavaScript logic for calculator operations
- `style.css` - Styling and responsive design

## Usage

1. Open `calculator.html` in a web browser
2. Click buttons to perform calculations or use your keyboard
3. Press `Esc` to clear the display

## Browser Support

Works in all modern browsers that support ES6 JavaScript and CSS Grid/Flexbox.
